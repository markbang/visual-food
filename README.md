# visual-food

可视化项目  团队：瓜头鲸

大众点评的店铺数据分析与可视化

>网站基于vue3 + [vue-echarts](https://github.com/ecomfe/vue-echarts)

>数据分析基于Python, MySQL和SPSS

# 项目结构
```bash
├── spider
│   ├── main.py      //爬虫主程序
├── src
│   ├── assets       //静态资源（包含处理过的数据）
│   ├── components   //图表组件
├── analysis
│   ├── data         //数据
│   ├── main         //分析代码
```
